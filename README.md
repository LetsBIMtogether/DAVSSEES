**D**elete **A**ll **V**iews, **S**heets, **S**chedules, **E**tc. **E**xcept **S**election

DAVSSEES is a Revit utility for fast removal of views, legends, sheets, schedules, etc.  
The user selects which drawings to keep and removes everything else in a single operation.  
The plugin provides progress feedback and summary logs after execution.

## Features

- Deletes all views, sheets, schedules, etc.
  - (Except for selected items you choose to keep.)
- Modeless WinForms.
- Text filter selection.
- Progress bar with cancel button.
- Success & error logs.
